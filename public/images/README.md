# Images Directory

This directory is for storing beach house images.

## Recommended Structure

```
images/
├── exterior/
│   ├── front-view.jpg
│   ├── side-view.jpg
│   ├── beach-access.jpg
│   └── sunset-view.jpg
├── interior/
│   ├── living-room.jpg
│   ├── kitchen.jpg
│   ├── bedroom-1.jpg
│   ├── bedroom-2.jpg
│   ├── bedroom-3.jpg
│   ├── bedroom-4.jpg
│   ├── bathroom-1.jpg
│   └── dining-area.jpg
├── amenities/
│   ├── pool.jpg
│   ├── terrace.jpg
│   ├── outdoor-seating.jpg
│   └── parking.jpg
└── views/
    ├── beach.jpg
    ├── ocean.jpg
    ├── sunrise.jpg
    └── sunset.jpg
```

## Image Guidelines

- **Format**: JPG or WebP
- **Resolution**: 1920x1080 or higher
- **File Size**: Optimize to under 500KB per image
- **Naming**: Use descriptive, lowercase names with hyphens

## Tools for Optimization

- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)
- ImageOptim (Mac)
- GIMP (Free, open-source)
